Getting Started with Go

This is a sample Go application. It's a reasonably simple app - but a good foundation for understanding how to get the most out of the platform.

About us Source on GitHub
To deploy your own copy, fork this repo and navigate through the code to learn the fundamentals of the platform [V5].

How this sample app works

  • This app was deployed to Koli. V2
  • When Koli received the source code it used the go tool chain to compile the application along with any vendored dependencies and created a deployable slug.
  • The platform then spins up a lightweight container that provides an isolated environment in which the slug can be mounted and executed.
  • You can scale your app, manage it, and deploy it from the Dashboard.
